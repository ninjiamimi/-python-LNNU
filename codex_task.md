## 十、使用 p4 作为基础内容页面背景板

当前页面两侧存在空白区域，希望用 p4 作为背景板。

请把 p4 图片放到：

```text
assets/basics-bg.jpg
```

使用位置：

- 基础内容独立页面的大背景
- 或基础内容页面左右空白区域

注意：

1. p4 是氛围背景，不是主视觉。
2. 不要直接高清贴图。
3. 必须做渐变和朦胧化处理。
4. 不要影响主体卡片文字和代码阅读。
5. 主内容卡片仍然保持白色 / 半透明白色。

视觉要求：

- 模糊
- 低饱和
- 低对比
- 偏冷灰蓝 / 雾白
- 从边缘向内容区透明渐变
- 像隔着玻璃或水面看背景

推荐 CSS：

```css
.basics-page {
  position: relative;
  min-height: 100vh;
  overflow-x: hidden;
}

.basics-page::before {
  content: "";
  position: fixed;
  inset: 0;
  pointer-events: none;
  background:
    linear-gradient(
      90deg,
      rgba(248, 250, 252, 0.12) 0%,
      rgba(248, 250, 252, 0.72) 35%,
      rgba(248, 250, 252, 0.96) 62%,
      rgba(248, 250, 252, 1) 100%
    ),
    url("./assets/basics-bg.jpg");
  background-size: cover;
  background-position: left center;
  background-repeat: no-repeat;
  filter: blur(10px) saturate(0.65) brightness(1.05) contrast(0.82);
  opacity: 0.68;
  z-index: 0;
}

.basics-page > * {
  position: relative;
  z-index: 1;
}
```

如果效果太糊，可以把 blur 调整为 `6px`。

如果图片太抢眼，可以把 opacity 调整为 `0.45 - 0.6`。

如果右侧过白，可以加一个非常淡的 `radial-gradient` 氛围光，但不要影响阅读。

注意：

不要把 p4 背景直接挂到 `body` 上，避免污染题目页和其他页面。

背景只属于 `basics-page`。
